Introduction

Welcome to my Model Predictive Control (MPC) project page. This repository contains comprehensive coursework and projects for the MPC module at KTH Royal Institute of Technology. The work covers a range of control strategies—from state feedback and LQR designs to advanced Nonlinear MPC for space rendezvous maneuvers using the Astrobee robot. The project integrates theoretical control principles with practical implementation using Python, CVXPY, and MOSEK.

Project Overview

State Feedback Control Design

This assignment focuses on modeling the Astrobee’s dynamics, discretizing the continuous-time model, and designing a state feedback controller with integral action to ensure zero steady-state error.

Finite-Time Optimal Control

In this task, a finite-time optimal control strategy is developed to perform rendezvous maneuvers. The assignment covers the discretization of dynamics, formulation of an optimal control problem, and constraints ensuring safe approach of the Astrobee robot.

Linear Quadratic Regulator

This assignment deals with designing and tuning an LQR controller. Topics include stability analysis, pole placement, and the effects of tuning the noise covariance matrices in an LQG framework.

Assignment 4: MPC Implementation

Here, the focus is on the practical implementation of a linear MPC controller. The assignment explores the influence of control horizons, the formulation of cost functions, constraint handling, and the impact of relaxing control limits.

Final Project: Nonlinear MPC for Space Rendezvous

The final project extends the coursework into a full-scale implementation of Nonlinear MPC. It involves modeling the nonlinear dynamics of the Astrobee robot, implementing time-varying reference tracking, and comparing various parameter sets to ensure robust rendezvous performance.

Theory and Methods Applied

The foundation of this project is built on key theoretical principles and practical control strategies:

  • Astrobee Dynamics: Modeling using Newton-Euler equations and quaternion kinematics for robust attitude representation.
  • State Feedback and LQR: Techniques to stabilize the system and minimize quadratic cost functions while ensuring reachability and stabilizability.
  • Finite-Time Optimal Control: Formulation of an optimal control problem to guarantee safe and efficient rendezvous maneuvers under constraints.
  • Model Predictive Control (MPC): Designing a receding-horizon optimization framework that incorporates state and control constraints to dynamically adapt to disturbances.
  • Nonlinear MPC (NMPC): Extending the MPC approach to handle nonlinear dynamics and time-varying references, ensuring robust performance even in the presence of uncertainties or actuator faults.

Advanced numerical solvers such as MOSEK are used together with Python-based optimization libraries (CVXPY) to solve these complex control problems in real time.

Results and Analysis

Extensive simulations were conducted to evaluate the performance of the controllers:

  • Control Horizon Analysis: Investigating how varying the prediction horizon affects the size of invariant sets and overall system performance.
  • Parameter Tuning: Comparison among different tuning sets (P1, P2, and P3) to balance convergence speed, energy consumption, and control input aggressiveness.
  • Disturbance Rejection: Evaluating system resilience under ideal conditions versus scenarios with faulty thrusters and external disturbances.
  • Tracking Performance: Assessment of both linear and nonlinear MPC strategies to achieve accurate rendezvous maneuvers with minimal tracking error.

The simulation results confirm that the designed controllers achieve robust performance, with precise state convergence and effective disturbance rejection.
